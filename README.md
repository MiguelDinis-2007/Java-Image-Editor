# Image Editor Java Project from 1st Year ISCTE

An Image Editor application developed in Java as a 1st-year academic project for ISCTE-IUL. Built using core programming logic and an internal university library, this project highlights my foundational understanding of application structure, object-oriented workflows, and practical coding execution.

## 🚀 Features
* **Basic Editing:** Invert, Contrast, Darken/Brighten, Margin
* **Advanced Filters:** Grain, Vignette, Retro Filter, Ancient Filter
* **Transformations:** Mirror (Vertical/Horizontal), Rotate, Crop, Expand
* **Utilities:** Copy, Cut, Paste, Undo

## 🛠️ Technologies Used
* **Language:** Java
* **Libraries:** ISCTE custom GUI library ("pt.iscte.greyditor")

## ⚙️ How to Run Locally
1. Clone this repository to your local machine.
2. Open the project in an IDE like IntelliJ IDEA or Eclipse.
3. Ensure the ".jar" file inside the "lib/" folder is added to your Project Structure as a library.
4. Run the main method to launch the editor interface!

> **Note:** The `monalisa.jpg` file is included in the root directory to be used as a test image.
